## Omni-2048
- This is my own version of the game 2048.
- The aim of this project is to implement versions of 2048 in a wide variety of programming languages and port it to a wide variety of platforms.

### How to play
- Slide numbered tiles up, left, right, and down.
- Ajacent matching tiles will combine into a tile with a number double the previous matching numbers.
- Every time you combine tiles, you gain points equal to the value of the new tile created.
- A new 2 or 4 tile appears in a random open position every time you slide or combine tiles.  This does not yield any points.
- Your goal is to earn as many points as possible and create a tile with a value of at least 2048.
- If the grid is full and you can't combine any more tiles, the game ends.
- The means by which you slide tiles varies by platform and may include w/a/s/d or arrow key presses, mouse clicks, swiping, or joystick/D-pad.
